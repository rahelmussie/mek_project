Adding an HTML Tile to a Page

Version 4
Created by yuval.z on May 20, 2015 3:53 PM. Last modified by yuval.z on Jun 29, 2015 9:00 AM.

    Starting with the 2015.2 Jive Cloud release, you can create a tile using your own custom HTML — without even the need to create your own add-on. You add an HTML tile to a page the same way you add other tiles: by managing a Jive place or editing a page, and selecting Add a tile.

    Note that there are some limitations to what you can put in this HTML tile. Refer to Custom HTML in Your Jive Community for more information.

     

    To add an HTML tile to a page:

     

    1. Be sure that your community manager has installed Jive's HTML Tile add-on from the global registry. See Installing an Add-on from the Global Registry for more information.
    2. Manage a place or edit a page so that you can add a new tile.
    3. Click Add a tile.
    4. Select the Create New Tile category.
    5. Choose Create an HTML Tile.
    6. Specify your HTML in the HTML field.
    7. Add images or other files using the Manage Files button.

      You can get the full URL for an image by clicking the "click to view URL" link. Use this URL in your HTML. For example:
      1. <img src="my.domain.com/resources/statics/1006/generic-html-tile-icon-128.png?a=1432741908331" />  
      <img src="my.domain.com/resources/statics/1006/generic-html-tile-icon-128.png?a=1432741908331" />
    8. Specify presentation settings, such as height or responsiveness, as necessary.
    9. Click Submit.
    10. Save changes to the page.

     

    Your HTML tile is now visible.

                    • Ben Zweig Dec 4, 2015 11:39 AM

                      Some good comments here. Summarizing:

                       

                      • Jive does not currently publish an installable HTML add-on for Hosted on On-Premise 8.0 communities, but it's in demand. I am currently working with my team to build one.
                      • The code is sandboxed to the tile itself, so you cannot affect other elements on the page. This safety measure is one reason tiles were created in the first place.
                      • Allegedly, only Administrators can add scripts to HTML tiles. But in practice, regardless of my permission level, I cannot add scripts to an HTML tile without seeing an error.
                      • The height must be hardcoded and cannot be resized dynamically. There is also no separate configuration for mobile height, which should be different in many use cases.

                                                            • mattdickens Aug 16, 2016 7:26 AM (in response to Ben Zweig)

                                                              Assuming you have the rights to include JS, try wrapping everything in a div and then add the following to the head of your html tile. In this instance the id of the wrapper is 'mainwrapper'. I added 100px in the example but you can adjust that as necessary.

                                                               

                                                              <script>

                                                              window.parent.$j(window).on("orientationchange",function(){

                                                                window.parent.$j(window.frameElement).height(document.getElementById('mainwrapper').offsetHeight+100);

                                                              });

                                                               

                                                              window.parent.$j(function() {

                                                              window.parent.$j(window.frameElement).height(document.getElementById('mainwrapper').offsetHeight+100);

                                                              });

                                                              </script>
